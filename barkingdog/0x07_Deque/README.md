# Deque

> Restricted Structure의 일종이며 Double Ended Queue의 약자로, 양쪽 끝에서 삽입과 삭제가 모두 가능한 자료구조이다.

## Properties of Deque

1. 원소의 추가 O(1)
2. 원소의 제거 O(1)
3. 제일 앞/뒤의 원소 확인 O(1)
4. 제일 앞/뒤가 아닌 나머지 원소들의 확인/변경이 원칙적으로 불가능

> Stack과 Queue와는 다르게 STL Deque에서는 index로 접근이 가능하다.
