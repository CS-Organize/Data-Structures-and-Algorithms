# [2751](https://www.acmicpc.net/problem/2751)
Created on: 2023-11-25

이 문제의 경우 시간제한이 1초이다. 따라서 O(n^2)의 알고리즘을 사용하면 시간초과가 발생한다. 따라서 O(nlogn)의 알고리즘을 사용해야 한다.
퀵정렬의 경우 최악의 경우 O(n^2)이지만, 머지정렬의 경우 최악의 경우 O(nlogn)이다.
